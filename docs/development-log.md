## Local Git Policy

- `main` 브랜치에서 로컬 commit으로 체크포인트를 남긴다.
- 원격이 연결되어 있으면 안정화된 commit을 push한다.
- 큰 기능 단위나 안정화가 끝날 때 `npm run build` 통과 후 commit한다.
- 진행 상황은 이 파일과 Git log를 함께 본다.
