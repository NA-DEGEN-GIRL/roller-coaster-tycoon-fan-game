# RCT Three.js Prototype

Three.js 기반의 RollerCoaster Tycoon 스타일 3D 프로토타입입니다. 직접 트랙을 조립하는 코스터 빌더보다, 만들어진 놀이기구 세트를 배치하고 길과 손님 동선을 관리하는 방향으로 잡았습니다.

## Run

```bash
npm install
npm run dev
```

## Current Scope

- Vite + TypeScript + Three.js 기본 구성
- Orthographic isometric camera
- 한국어 기본 UI와 영어 전환 언어 선택
- 전체 평지 타일형 지형
- 클릭 가능한 공원 메인 입구와 open/close 운영 상태
- `src/gameConfig.ts`에서 조절 가능한 시작 예산, 건설 비용, 입장료/탑승료 기본값
- 길, 잔잔한 놀이기구, 꾸미기로 나뉜 건설 카탈로그
- 카테고리 탭을 누르면 항목 팝업이 열리는 건설 UI
- 길 설치와 철거
- 대기줄 길 설치와 철거
- 길/대기줄 드래그 연속 설치
- 대상 종류를 잠그는 불도저 드래그 삭제
- 길이 끊긴 손님이 주변을 헤매다가 가까운 길로 복귀하는 seeking 동작
- 손님별 가격 민감도/인내심/취향 기반 목적지 선택과 BFS 이동
- 공원 입장료와 놀이기구 탑승료 설정, 입장/탑승 수익 반영
- 설치 순서 기반 대기줄 연결과 손님 입장 지점 지정
- 회전목마 prefab 설치
- 회전목마 설치 전 3D 프리뷰와 R 키 4방향 회전 배치
- 회전목마 open/closed 상태
- 회전목마별 탑승료와 손님 탑승 가치 판단
- 회전목마 입구/출구 설치
- 회전목마 설치 후 입구/출구 배치로 이어지는 선택 패널 플로우
- 선택한 놀이기구가 있을 때만 표시되는 놀이기구 조작 패널
- 회전목마 로딩/운행/하차 상태
- Web Audio API 합성 회전목마 음악, 음악 프리셋 선택, 거리 기반 볼륨, 음악 on/off
- 전나무와 벚꽃 나무 배치
- 잔디 타일 안에서 위치/회전/크기를 조절해 여러 그루를 놓을 수 있는 나무 장식
- 일반 길 외곽에 설치되어 손님 이동을 막지 않는 벤치 장식
- 피곤한 손님이 빈 벤치를 찾아가 앉아 쉬는 행동
- `model-preview.html`에서 손님 모델의 서기/걷기/앉기 pose를 독립 확인
- 공원 입구에서 들어오고, 공원 close 시 입구로 퇴장하는 손님
- 손님 선택 팝업, 손님별 번호/상태 수치, 선택 손님 카메라 추적
- 실제 오디오 루프 파일을 사용하는 군중 ambience 믹싱
- 하단 debug 콘솔
- 휠 확대/축소, Q/E 쿼터뷰 회전, 연속 Q/E 회전 옵션, WASD 카메라 이동
- 건설 중 `Esc`로 선택 툴 복귀
- 디버그용 시뮬레이션 배속 버튼
- 기본 선택/빌드 툴 패널과 하단 상태 바
- 모바일/데스크톱 반응형 레이아웃
